# serverless-ts-monorepo-sample

Sample code of Yarn Workspaces with ServerlessFramework.

## Dependencies

- Node.js v14.15.1
- Yarn v1.22.5
- ServerlessFramework v2.52.1

## Deployment

```
yarn

cd packages/hello
sls deploy

cd ../packages/bye
sls deploy
```
